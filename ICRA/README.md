# ICRA Paper Lists

Accepted-paper lists for the **ICRA** (IEEE International Conference on Robotics
and Automation) conference. For each year there is a Markdown table
(`ICRA<YEAR>_Paper_List.md`) for quick browsing and a
`ICRA<YEAR>_Paper_List_with_Abstract.csv` with the full records including
abstracts.

## Sources & credits

The data in this directory is collected from the official ICRA conference
program pages and is sourced from the following community-maintained paper-list
